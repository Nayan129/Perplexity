# 🧠 Neurovia AI

Neurovia AI is an intelligent chat application designed to simulate natural thinking and conversational flow. The name "Neurovia" represents the pathway of intelligence — where ideas, questions, and answers connect seamlessly.

This application delivers a modern AI chat experience inspired by tools like ChatGPT and Perplexity, with a focus on performance, usability, and clean design.

---

## 🚀 Features

* 💬 Real-time AI-powered chat
* 🧠 Smart chat title generation
* 📂 Persistent chat history
* ⚡ AI typing/loading indicator
* 🧾 Markdown rendering (code, lists, etc.)
* 📱 Fully responsive UI (mobile + desktop)
* 🗂 Sidebar chat navigation
* 🗑 Chat deletion with confirmation
* 🎯 Smooth and minimal user experience

## ⚙️ Installation

```bash
git clone https://github.com/your-username/neurovia-ai.git
cd neurovia-ai
```

### Install dependencies

```bash
cd client && npm install
cd ../server && npm install
```

---

## 🔧 Environment Variables

Create `.env` in server:

```env
PORT=3000
MONGO_URI=your_mongodb_uri
```

---

## ▶️ Run Project

```bash
# backend
npm run server

# frontend
npm run dev
```
